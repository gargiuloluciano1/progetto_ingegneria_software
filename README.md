## Punti da definire: Applicazione Desktop o Web?
Per una stazione sciistica la scelta piu sensata sarebbe un applicazione desktop (x Windows)
Dovremmo trovare qualcosa di simile a JavaFX


API/Framework possibili per un applicazione __desktop__:
    0. WindowPresentationFramework WPF 
        Sembra essere il piu consigliato, barebone
        Parte grafica con XML, eventi gestiti tramite C# 
    1. GTK#, 
        Wrapper per gtk, Multipiattaforma, documentato
        Sintassi complessa
    2. .NetMAUI
        
    3. Avalonia

API/Framework possibili per un applicazione __web__: 
    1. ASP.Net Core
    2. Razor
